# Test App

## Getting Started

```bash
cd packages/test-app
yarn install
yarn dev
```

Open `http://localhost:3001` in the browser.

## Page Structure

- **Page A** (`/`): Contains a Link component to navigate to Page B.
- **Page B** (`/b`): Contains two buttons that update the query parameter to `test=first` or `test=second` when clicked.

## Version Information

- Next.js: 15.5.6
- React: 19.2.0
- React DOM: 19.2.0
